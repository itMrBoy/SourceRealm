# llmdoc Index

This directory is the stable project memory for SourceRealm. Temporary research belongs under `.llmdoc-tmp/` and is not part of the stable doc set.

## Startup

- `llmdoc/startup.md` - first file to read when starting work in this repository.

## Must Read

- `llmdoc/must/project-basics.md` - product, package layout, commands, data directories.
- `llmdoc/must/current-state.md` - current implementation state and known naming risk.
- `llmdoc/must/working-agreement.md` - local conventions for changes and validation.

## Overview

- `llmdoc/overview/project-overview.md` - high-level product and architecture overview.

## Architecture

- `llmdoc/architecture/server-api-and-generation.md` - Fastify API, generation, storage, and provider flow.
- `llmdoc/architecture/incremental-update.md` - git diff update pipeline and state semantics.
- `llmdoc/architecture/frontend-runtime.md` - React screens, API client, map flow, and level runtime.

## Reference

- `llmdoc/reference/data-model.md` - shared schemas, stored JSON files, task types, scoring, and judging semantics.
- `llmdoc/reference/commands.md` - common repo commands and environment variables.

## Memory

- `llmdoc/memory/reflections/` - future reflections, not stable architecture docs.
- `llmdoc/memory/decisions/` - future decision records, not startup-required docs unless indexed later.
