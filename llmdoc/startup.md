Then route by task:

- Git diff, stale/obsolete levels, or "world changed" behavior: `llmdoc/architecture/incremental-update.md`
- React screens, map/level UX, task runtime, or frontend API calls: `llmdoc/architecture/frontend-runtime.md`
- Schemas, task shapes, progress, badges, scoring: `llmdoc/reference/data-model.md`
- Commands and environment variables: `llmdoc/reference/commands.md`

Before editing, check for the current package naming mismatch described in `llmdoc/must/current-state.md`.
